Computer Graphics I - Programming Assignment 2: 2D Drawing

Implemented using JavaScript, HTML5, CSS3 and some use of Bootstrap.

Objective: Implemented question 1 and 2 of Exam 1.

Project provides the interactive UI. User must provide input like (x, y) point, score, etc to draw particular Pattern.

Question 1 is about drawing fractal of particular shape. I was able to implement only line fractal.

Question 2 is about drawing wheel shape based on driver's score.

Open assignment2.html in any browser to see application.
